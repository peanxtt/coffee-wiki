The Dialing-In Protocol

01.

Identify Base Variable

Lock your water temperature (94°C) and ratio (1:16). Only adjust grind size during initial calibration.

02.

Evaluate Resistance

Monitor flow rate. A "hollow" flavor indicates channeling, while a heavy, bitter finish suggests over-extraction.

03.

Analyze TDS Threshold

Measure the solubles. If TDS is below 1.2%, tighten the grind. If above 1.5%, increase the coarseness.
