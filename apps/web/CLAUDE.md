# apps/web/

## Files

| File                 | What                                 | When to read                                          |
| -------------------- | ------------------------------------ | ----------------------------------------------------- |
| `package.json`       | Next.js app dependencies and scripts | Adding dependencies, configuring app scripts          |
| `next.config.js`     | Next.js configuration                | Modifying build settings, adding plugins, env config  |
| `tailwind.config.ts` | Tailwind CSS configuration           | Customizing theme, adding colors, extending utilities |
| `tsconfig.json`      | TypeScript configuration             | Modifying compiler options, path aliases              |
| `postcss.config.js`  | PostCSS configuration                | Adding CSS processors, configuring Tailwind           |
| `.eslintrc.json`     | ESLint rules for web app             | Modifying linting rules, adding custom rules          |
| `.gitignore`         | Git ignore patterns for web app      | Adding app-specific files to ignore                   |

## Subdirectories

| Directory | What                          | When to read                                            |
| --------- | ----------------------------- | ------------------------------------------------------- |
| `src/`    | Application source code       | Implementing features, pages, components, business logic|
| `public/` | Static assets (images, fonts) | Adding static files, favicon, manifest                  |
