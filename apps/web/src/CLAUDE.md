# apps/web/src/

## Files

| File           | What                              | When to read                                        |
| -------------- | --------------------------------- | --------------------------------------------------- |
| `globals.css`  | Global CSS styles and Tailwind    | Modifying global styles, adding custom CSS          |

## Subdirectories

| Directory      | What                              | When to read                                        |
| -------------- | --------------------------------- | --------------------------------------------------- |
| `app/`         | Next.js App Router pages          | Creating routes, layouts, implementing pages        |
| `components/`  | React components                  | Building UI components, reusable elements           |
| `hooks/`       | Custom React hooks                | Creating reusable logic, state management hooks     |
| `lib/`         | Utility functions and helpers     | Adding utilities, third-party integrations          |
| `stores/`      | Zustand state stores              | Managing global state, creating state slices        |
