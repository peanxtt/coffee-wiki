# coffee-wiki-design/

Static HTML design prototypes and mockups for The Global Grind & Whisk. These serve as visual references for implementing the production Next.js application.

## Subdirectories

| Directory              | What                                | When to read                                      |
| ---------------------- | ----------------------------------- | ------------------------------------------------- |
| `matcha_region_page_1/`| Home/landing page prototype         | Implementing landing page, hero, navigation       |
| `matcha_region_page_2/`| Secondary page variant              | Exploring alternative layouts                     |
| `matcha_region_page_3/`| Coffee brewing guide variant        | Implementing brewing guides                       |
| `matcha_region_page_4/`| Matcha cultivar page                | Building cultivar detail pages                    |
| `matcha_region_page_5/`| Uji, Kyoto region profile           | Implementing region profile pages                 |
| `matcha_region_page_6/`| Alternative region layout           | Exploring region page variations                  |
| `matcha_region_page_7/`| Coffee origins page                 | Building coffee origin pages                      |
| `matcha_region_page_8/`| Detailed brewing page               | Implementing detailed brewing instructions        |
| `matcha_region_page_9/`| Encyclopedia index                  | Building index/listing pages                      |
| `matcha_region_page_10/`| Brewing & Prep hub                 | Implementing brewing methods overview             |

Each directory contains:
- `code.html` - HTML prototype with Tailwind CSS
- `screen.png` - Screenshot of the design

These prototypes use:
- Tailwind CSS (CDN)
- Material Symbols Outlined icons
- Fonts: Plus Jakarta Sans, Playfair Display, Lora
- Color palette: Coffee browns, matcha greens, cream backgrounds
