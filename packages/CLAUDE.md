# packages/

## Subdirectories

| Directory   | What                                    | When to read                                          |
| ----------- | --------------------------------------- | ----------------------------------------------------- |
| `ui/`       | Shared UI components (Tamagui)          | Creating reusable components, building design system  |
| `database/` | Supabase client and database types      | Database operations, auth, querying data              |
| `utils/`    | Shared utility functions                | Using formatting, validation, general utilities       |
| `config/`   | Shared configurations (ESLint, TS, etc.)| Extending configs, setting up new packages            |
